Comandos Git 
git status
 → mostra tudo que foi alterado

 git add -A 
 → adiciona essas alterações ao git para que possam ser comitadas

 git commit -m "texto descritivo"
 → persist as alterações 

  git commit -am "texto descritivo"
 → com o am ele ja adiciona e comita persist as alterações 

 git branch
 → lista todos os branch e os com * é o que estamos trabalhando 

 git brand 
 → lista todos os branchs da sua pasta

 git checkou nome do branch
 → redireciona para o novo branch 

 git reste {soft, mixed , hard}
 → soft voltar antes do commit, mas ainda com as alteraçoes realizadas
 mixed parecido com o soft porem ainda sem o add -A
 hard apaga todo o commit
